# Resume Parser using NLP

A Python-based Resume Parser that automatically extracts important information such as name, email, phone number, and skills using Natural Language Processing techniques.

## Features
- Resume text extraction
- Email and phone number detection
- Skill identification using keyword matching
- Clean and readable output

## Technologies Used
- Python
- NLP (Text Processing)
- Regular Expressions

## How It Works
1. Resume text is read from a file
2. NLP-based pattern matching extracts key details
3. Parsed information is displayed in structured format

## How to Run
```bash
python resume_parser.py
